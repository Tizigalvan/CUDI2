Manual del Programador - Sistema CUDI
1. Visión General
El Sistema de Gestión de Disposición Aúlica CUDI es una aplicación web diseñada para administrar la asignación de aulas, horarios y recursos educativos en una institución académica.

2. Estructura del Proyecto
2.1 Directorios Principales
/bd/: Contiene los scripts SQL de la base de datos
/css/: Hojas de estilo
/img/: Imágenes y recursos gráficos
2.2 Archivos Principales
disposicionaulica.php
: Página principal del sistema
conexion.php
: Configuración de conexión a la base de datos
config_timezone.php
: Configuración de zona horaria
añadir_aula.php: Gestión de aulas
gestionar_materias.php
: Administración de materias
gestionar_profesores.php
: Gestión de profesores
3. Base de Datos
3.1 Tablas Principales
aulas
id_aula: Identificador único
piso: Número de piso
cantidad: Capacidad de estudiantes
numero: Número/identificador del aula
materias
id_materia: Identificador único
nombre: Nombre de la materia
carrera_id: Referencia a la carrera
profesor_id: Referencia al profesor
tarjetas_disposicion
id_tarjeta: Identificador único
fecha: Fecha de la clase
turno_id: Referencia al turno
itinerario_id: Referencia al horario
materia_id: Referencia a la materia
aula_id: Referencia al aula
profesor_id: Referencia al profesor
estado: Estado de la tarjeta
